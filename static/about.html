What is InfiniX?

InfiniX is a website I wrote when I was on Junior High. Back then, this was a project I needed to write for our HTML-CSS related subjects. The goal of this rewrite is to now use more modern technologies to implement an existing codebase. This would include writing a backend that will be hosted on a server (hopefully).

The current goal of this website is to be 'basically' a blog; showcasing the games I truly like. This would of course include games I may write in the future.
